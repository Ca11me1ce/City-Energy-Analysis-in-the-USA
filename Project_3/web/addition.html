Text Analysis

A topic model is a type of statistical model for discovering the abstract topics that occur in a collection of documents. That can help the analysis to get more relative ideas with the current text. The recent academic papers are colleced from google scholar and CNKI according to the keywords - electricity consumption or natural gas consumption. The number of papers is 100+.

This project implements the LDA (Latent Dirichlet Allocation) algorithm to investigate the prevalent topics about energy consumptions. The result shows that popular topics contain “energy combustion”, “CO2”, ”renewable”, “study approaches”. In other words, pollution, renewable energy, and study methods are prevalent topics when researchers implement energy data analysis.

Network Analysis

Network analysis can inform the analysis for developing practical ways to ensure an effective flow of information and resources. That can present the relationships between every node in the network more intensively.

To investigate the relationship between energy consumption and greenhouse gas emission, we bin the variables - gas_mcf and gas_lb_ghg into 50 bins which represent the 50 different levels and store the binning number as nodes in the network. The weight is generated according to the count of rows with the same two nodes. Therefore, if a city has a natural gas consumption level A and greenhouse gas emission level B, the weight of A and B increments 1. We can identify the relationship between natural gas consumption and greenhouse gas emission according to the clusters generated.

This network shows that the strongest relationship is between the same levels. In other words, a city consumes a normal amount of natural gas, then it produces a normal amount of greenhouse gas.
